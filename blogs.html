Question-1: What are the different ways to select an element in the DOM?

By ID – getElementById, By Class – getElementsByClassName, By Tag Name – getElementsByTagName, By Name Attribute – getElementsByName, Using querySelector – Selects the first matching element, Using querySelectorAll – Selects all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML which gets or sets the HTML inside an element. innerText which Gets or sets only the visible text. textContent which Gets or sets all text inside an element.

Question-3: What is event delegation in the DOM?

Event delegation means adding an event listener to a parent element instead of its children. When a child is clicked, the event "bubbles up" to the parent, which handles it. This is useful for managing many elements efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and moves up to its parent, grandparent, and so on. It allows parent elements to listen for events on their children.

Question-5: How do you create, add, and remove elements using JavaScript?

We can Create an element using document.createElement("tag"), then Add: parent.appendChild(child) & then removing it with Remove: parent.removeChild(child).
